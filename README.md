# Marketing Campaign Performance Dashboard

## Project Overview

This project analyzes marketing campaign performance across multiple channels such as Google Ads, Facebook, Instagram, LinkedIn, Email, and YouTube.

The dashboard helps marketing teams evaluate ROI, CTR, conversion rates, revenue generation, and customer acquisition efficiency.


## Tools Used

- Power BI
- DAX
- Excel / CSV
- Power Query

## Key KPIs

- Total Marketing Spend
- Total Revenue Generated
- ROI %
- CTR %
- Conversion Rate %
- Cost Per Click (CPC)
- Customer Acquisition Cost (CAC)

## Dashboard Features

- KPI Cards
- Campaign ROI Analysis
- Revenue by Campaign
- Platform Performance Comparison
- Conversion Funnel Analysis
- Audience Segment Analysis
- Regional Performance Analysis


## Key Insights

- Identified highest ROI campaigns
- Compared marketing channel performance
- Evaluated customer acquisition efficiency
- Analyzed regional profitability
- Optimized marketing budget allocation
